ENSEIRBprojetS9sujet6
=====================

Commande d'un Webmail par le geste ou la voix (Enseirb - 3A - Projet S9 - Sujet 6) : 

L'objectif du projet est de réaliser plusieurs extensions sur des navigateurs récents permettant 
de commander un webmail par le geste ou la voix. Nous ciblons en particulier le 
développement d’extensions implémentant les librairies Javascript permettant de contrôler 
une kinect ou un leapmotion controller ou bien encore l'API Speech Input (disponible en tant 
que balise html5 sur Chrome et permettre la saisie de commande vocale).
